Class Properties
select-none user-select: none;
select-auto user-select: auto;
select-text user-select: text;
select-contain user-select: contain;
select-all user-select: all;

None

You can apply "select-none" to the object's class.
Prevents the text of this element and the elements below it from being selected.

Never underestimate your own ignorance.

<p class="select-none ... "> Never underestimate your own ignorance. </p>

Auto

You can apply "select-auto" to the object's class.
The used value of auto is determined as follows:

  • On the ::before and ::after pseudo elements, the used value is none
  • If the element is an editable element, the used value is contain
  • Otherwise, if the used value of user-select on the parent of this element is all, the used value is all
  • Otherwise, if the used value of user-select on the parent of this element is none, the used value is none
  • Otherwise, the used value is text

It's not that I'm so smart , it's just that I stay with problems longer.

<p class="select-auto ... "> It's not that I'm so smart , it's just that I stay with problems longer. </p>

Text

You can apply "select-text" to the object's class.
The text can be selected by the user.

Wisdom is not a product of schooling, but of the life- long attempt to acquire it.

<p class="select-text ... "> Wisdom is not a product of schooling, but of the life- long attempt to acquire it. </p>

Contain

You can apply "select-contain" to the object's class.
Enables selection to start within the element; however, the selection will be contained by the bounds of that element.

We still do not know one - thousandth of one percent of what nature has revealed to us.

<p class="select-contain ... "> We still do not know one - thousandth of one percent of what nature has revealed to us. </p>

All

You can apply "select-all" to the object's class.
The content of the element shall be selected atomically: If a selection would contain part of the element, then the selection must contain the entire element including all its descendants. If a double-click or context-click occurred in sub-elements, the highest ancestor with this value will be selected.

Let every man be respected as an individual and no man idolized.

<p class="select-all ... "> Let every man be respected as an individual and no man idolized. </p>

All examples are quotes from Albert Einstein
