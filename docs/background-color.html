use

You can apply and apply "bg-{color}" to the object's class.

Button
<a> class="btn bg-yellow-400">Button</a>

hover

Put "hover:" before the object class.

Button
<a> class="btn bg-green-400 hover:bg-green-500">Button</a>

active

Put "active:" before the object class.

Button
<a> class="btn bg-purple-400 hover:bg-purple-500 active:bg-purple-600">Button</a>
