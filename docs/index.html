Introduction

videt is a web framework that helps you easily create the necessary elements for the web. It provides utility classes and designed classes, and when you want to create something, you just need to apply a class in HTML without CSS.

Quick start

To quickly add to your project, use jsDelivr, which offers a free CDN.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/brazens/videt@0.5.0/dist/videt.min.css">

How to use

You can create various things on the web with the utility class and design class provided by default.

button button button
<!-- Quick design using videot, examples -->
<a class="btn bg-purple-400">button</a>
<a class="btn bg-blue-400">button</a>
<a class="btn bg-green-400">button</a>
