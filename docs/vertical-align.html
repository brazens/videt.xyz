Vertical Alignment

This utility class specifies vertical alignment inline or in a table-cell box.

Class Properties
align-top vertical-align: top;
align-middle vertical-align: middle;
align-bottom vertical-align: bottom;
align-text-top vertical-align: text-top;
align-text-bottom vertical-align: text-bottom;
align-baseline vertical-align: baseline;
align-sub vertical-align: sub;
align-super vertical-align: super;

use

This can be used to align an element's box within a line of a text element when you want it to align vertically within the line box it contains.

top

You can apply "align-top" to the object class.

align : top

<span class="align-top ... "> ... </span>

middle

You can apply "align-middle" to the object class.
Align the middle of the element to the parent's baseline + x-height / 2.

align : middle

<span class="align-middle ... "> ... </span>

bottom

You can apply "align-bottom" to the object class.

align : bottom

<span class="align-bottom ... "> ... </span>

text top

You can apply "align-text-top" to the object class.
Aligns the top of the element with the top of the parent element's font.

align : text-top

<span class="align-text-top ... "> ... </span>

text bottom

You can apply "align-text-bottom" to the object class.
Aligns the bottom of the element with the bottom of the parent element's font.

align : text-bottom

<span class="align-text-bottom ... "> ... </span>

baseline

You can apply "align-baseline" to the object class.

align : baseline

<span class="align-baseline ... "> ... </span>

sub

You can apply "align-sub" to the object class.

align : sub

<span class="align-sub ... "> ... </span>

super

You can apply "align-super" to the object class.

align : super

<span class="align-super ... "> ... </span>
