Hover, Active

It can be used by adding a state transition prefix to the videt utility class name.

Hover

Just add "hover:" in front of it and you can use it.

Hover Button
<a class="btn bg-purple-500 hover:bg-purple-600 text-white">Hover Button</a>

It can be used in the following utility classes.

  • background colors
  • text color
  • SVG fill
  • border color
  • opacity
  • cursor

Active

Just add "active:" in front of it and you can use it.

Hover Button
<a class="btn bg-green-500 active:bg-green-600 text-white">Hover Button</a>

It can be used in the following utility classes.
