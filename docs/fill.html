use

You can apply and apply "fill-{color}" to the object's class.

<a> class="fill-yellow-400 ... "> ... </a>

hover

Put "hover:" before the object class.

<a> class="hover:fill-red-500 ... "> ... </a>

active

Put "active:" before the object class.

<a> class="active:fill-purple-600 ... "> ... </a>
